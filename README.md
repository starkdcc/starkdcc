## <img src="https://media.giphy.com/media/l2JJyLbhqCF4va86c/giphy.gif" width="28"/>&nbsp;&nbsp;Vibe Coder Detector™

> Score yourself honestly. 7+ checks = you're ngmi.

<table>
<tr><td>

**🚩 Red flags**

- [ ] You copy-paste from ChatGPT without reading
- [ ] Your variables are named `temp`, `temp2`, `final_temp`
- [ ] You write `except: pass` and call it error handling
- [ ] You push to `main` because "branches are for cowards"
- [ ] Your commits say `"update"`, `"fix"`, `"asdf"`
- [ ] You say *"it works on my machine"* unironically
- [ ] Your last README says `# Project`
- [ ] You think `time.sleep(5)` is a valid retry strategy
- [ ] You wrap everything in `try/except Exception`
- [ ] Your code has 47 TODOs from 2022
- [ ] You asked AI "is this code good?" instead of reading it
- [ ] You pushed `.env` to GitHub (at least once)

</td><td>

**✅ Certified non-vibe**

- [ ] You read the stack trace before panicking
- [ ] Your commits are imperative: `add X`, `fix Y`
- [ ] You use `.gitignore` from muscle memory
- [ ] You write the test before the fix
- [ ] You know what `__init__.py` actually does
- [ ] You've debugged with `print()` and aren't ashamed
- [ ] You delete code more than you add
- [ ] You read docs over Stack Overflow
- [ ] Your function names describe what they do
- [ ] You handle errors specifically, not generically
- [ ] You pin versions in `requirements.txt`
- [ ] Your virtualenv is not named `venv1`, `venv2`, `venv_final`

</td></tr>
</table>

<div align="center">

<kbd>&nbsp;Score 0-3&nbsp;</kbd> &nbsp; **You actually write code.** Rare. Respect.

<kbd>&nbsp;Score 4-6&nbsp;</kbd> &nbsp; **Mid.** Touch grass, read a book, then come back.

<kbd>&nbsp;Score 7-10&nbsp;</kbd> &nbsp; **Certified vibe coder.** Your codebase is held together by prayers.

<kbd>&nbsp;Score 11+&nbsp;</kbd> &nbsp; **Straight up npm package collector.** Your `node_modules/` is a black hole.

</div>
