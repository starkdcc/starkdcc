## About

I build backend systems, crypto tooling, and Discord bots — mostly Python. I ship small tools that do one thing well, and larger projects when the tooling doesn't exist. Currently focused on **multi-chain escrow automation** and **wallet infrastructure**.

- Building stuff at **~/projects**
- Learning **Rust** and **Solidity** in spare cycles
- Ask me about **Discord.py**, **Web3.py**, **MongoDB aggregations**
- Reach me via GitHub issues or discussions
