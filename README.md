<details>
<summary><b>🤖 Auto MM Bot — Multi-chain Crypto Escrow</b></summary>

<br/>

> A production-grade Discord middleman bot for secure peer-to-peer crypto trading.

**Tech:** `Python` · `Discord.py` · `MongoDB` · `Web3.py` · `Solana` · `TronPy` · `BlockCypher`

**Features:**
- 🔒 **Isolated escrow wallets** — fresh wallet per deal
- 💰 **9 cryptocurrencies** — BTC, ETH, SOL, LTC, DOGE, USDT (ERC/TRC), BNB, POL, XRP
- 🎭 **Role-based deal flow** — Sender/Receiver with full state machine
- 🚨 **Dispute resolution** — admin-mediated refund/release
- ⭐ **Reputation & badges** — vouches, trust scores, 6-tier badges
- 🏆 **Leaderboards** — top traders by volume & vouches
- 📊 **Live stats** — server-wide analytics & revenue reports
- ⚡ **Auto-payouts** — instant release from escrow on confirmation
